ls -> Muestra los elementos.
cd -> Change directiry, nos movemos a una carpeta.
git -> Muestra ayuda de los comandos disponibles para git
git status -> Nos muestra los cambio que se han hecho en el repositorio
git add -A -> Preparamos los cambios que vamos a subir
git commit -m "mensaje" -> Asignamos un mensaje al cambio que agregamos anteriormente
git push origin master -> Subimos cambios a github
